LBGW - LoRa/ BLE Gateway

Product Summary

The LBGW is a Long Range (LoRa) / Bluetooth Low Energy (BLE) connectivity gateway that collects data from BLE sensors and LoRa sensor nodes, and sends data to the remote cloud server.

Specifications

Hardware Type
LoRa/BLE Gateway
Model #
LBGW

Features

  • Highly Efficient 32-bit Quad-core Processor
  • 1GB RAM/ 8GB ROM
  • Supports Cellular(2G/3G/4G), LoRa, Wi-Fi, Ethernet & BLE
  • Input Voltage: 5V/2A
  • Can connect Up to 25 LoRa devices with multiple gateways
  • Remote Configurable
  • Supports multiple protocols HTTP(s), TCP, MQTT TLS1.0
  • Battery Backup up to 5 Hours (Optional)
  • Encrypted data exchange between Gateway and cloud
